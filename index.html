Journey
  • My journey into the world of problem-solving really kicked off when I was just a kid. I loved tinkering around the house, whether it was fixing a leaky faucet or trying to beat my high score in strategy games. There’s something so satisfying about cracking a tough nut! But it wasn’t until secondary school that I found my true passion: coding.
Learning
  • Since then, I’ve explored Python, JavaScript, C++ and several other smaller languages and Frame-works to build things that interest me including a few web projects, apps and modifications. I’m currently working on learning React and also experimenting with Python for automating tasks. School projects and online tutorials have helped build my confidence.
Projects
  • I’ve explored programming through a range of small, hands-on projects. In C++, I built a calculator, a probability calculator, and experimented with basic API integration. With Python, I developed a few simple games, created a GUI-based currency converter using Tkinter, and built a basic banking system using Python and SQL. I’ve also used JavaScript to develop small web apps focused on experimenting with API's, including a weather application using the OpenWeather API and a currency converter using the ExchangeRate API.
