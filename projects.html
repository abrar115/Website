FAQ

Q. How do you download it?

A. Click Download at the top => click on code => click on Download Zip => after downloading, double click the zip => take the file out (it should be named the same thing as the zip is) => go to .minecraft folder => .config => .chattriggers => modules => paste it there

Q. How do i open gui?

A. /su gui || /su help for all commands

Q. It isnt working.

A. Make sure to be on CTJS 2.20 and if the issue still persists, check the download.

Q. How do i know this is safe?

A. A rat scanner, specific for CTJS can be used to verify its safety, a popular scanner is Rat Scanner Discord.

Q. Can I download this without the github?

A. This Module is verified on the official chattriggers website Chattriggers. It can be imported using the command "/ct import slayerutils" but keep in mind it could be outdated.
